# TEA sepolia testnet bot autosend

Clone repo
```shellscript
git clone https://github.com/edosetiawan-xyz/Sepolia-tea-multisender.git
cd Sepolia-tea-multisender
```

install dependencies
```shellscript
npm install
```

fill privatekey on .env
```shellscript
cp .env.example .env
nano .env
```

Run bot
```
node tea.js
```
